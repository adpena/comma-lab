## Wire format (format_id=0x02)

```
magic(1)=0xFE | format_id(1)=0x02 | pr106_len(4) | pr106_bytes(pr106_len)
| pr101_payload_len(2) | pr101_payload(pr101_payload_len)
| noop_count(2) | dim_bytes(2) | rank_bytes(1) | noop_rank_bytes(1)
```

The PR101 payload is the output of
`tac.packet_compiler.encode_ranked_no_op_sidecar` with schema
`(n_pairs=600, n_dims=28, deltas=(-2,-1,1,2), huff_min_len=2, huff_max_len=8,
no_op_sentinel=255)`. Format-id=0x01 (brotli) remains supported for backward
compatibility with the r2 r1 archive bytes.
