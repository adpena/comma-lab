ALWAYS save the complete output before summarizing.

as a reason to skip, delay, or deprioritize work. Implement everything for science
and the community. The user explicitly removed the time constraint.

when it returns, BEFORE any summarization or memory saving. The Downloads folder
is the permanent unsummarized archive. Format: council_TOPIC_YYYYMMDD.md

These three rules override ALL other guidance, including council recommendations
that conflict with them.
