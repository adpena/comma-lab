# L5 v2 Latest Source Basis Wire-In

Date: 2026-05-16
Author: codex
Scope: L5 v2 staircase, Cathedral literature anchors, paper-fidelity custody

## Finding

The paper-fidelity review found that the latest L5 source-refresh ledger named
PNVC, SNeRV, MetaNeRV, and C3 as actionable design prompts, but the canonical
`time_traveler_l5_v2` research-basis family did not yet carry those source
IDs. That made the Cathedral row less auditable: the row could describe
temporal INR or practical codec lessons without a machine-readable source
contract and proof gate.

## Fix

- Added `pnvc_2025` to the canonical research source registry.
- Added source aliases for `pnvc`, `snerv_spectra`, and `metanerv`.
- Added `pnvc_2025`, `snerv_spectra_2025`, `metanerv_2025`, and
  `c3_neural_compression_2024` to the `time_traveler_l5_v2` research-basis
  family.
- Updated the TT5L composition row literature and support text so the newer
  practical-INR, frequency-split, pretraining/curriculum, and decode-complexity
  signals are visible to Cathedral/autopilot without becoming score claims.

## Source Scope

These sources remain planning-only:

- PNVC supports co-designed representation + entropy + decode-speed discipline.
- SNeRV supports explicit frequency split / spectral-bias ablation.
- MetaNeRV supports pretraining and progressive fitting as a speed/curriculum
  hypothesis.
- C3 supports low-complexity overfitted neural compression and OSS/DX
  benchmarking.

None of the above is a Pact score claim. L5 v2 still must prove byte-closed
payloads, scorer-free inflate consumption, T4 decode timing, and paired
`[contest-CPU]` / `[contest-CUDA]` exact eval before promotion or submission.
