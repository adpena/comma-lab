## 3. Scorer architecture (verified facts the council relies on)

Per `upstream/modules.py` line-by-line read:

**SegNet pipeline:**
1. Input: `(B, T=2, C=3, H=874, W=1164)` RGB uint8
2. `preprocess_input`: select last frame `x[:, -1, ...]` — **frame 0 is invisible to SegNet**
3. Bilinear interpolate to `(384, 512)` → `(B, 3, 384, 512)`
4. `smp.Unet('tu-efficientnet_b2', classes=5)` with stride-2 stem → `(B, 5, 384, 512)`
5. Distortion = `(argmax(out_1) != argmax(out_2)).float().mean()` — DISCRETE argmax-disagreement
6. **Logit MAGNITUDES are irrelevant; only ORDERING matters**

**PoseNet pipeline:**
1. Input: pair `(B, T=2, C=3, H=874, W=1164)` RGB
2. Rearrange + bilinear to `(384, 512)` + `rgb_to_yuv6` → `(B*T, 6, 192, 256)` Bayer-pattern
3. Rearrange to `(B, T*6=12, 192, 256)`, normalize `(x - 127.5) / 63.75`
4. FastViT-T12 → 2048-dim → summarizer → ResBlock → Hydra → 12-dim pose
5. Distortion = `(out_1.first6 - out_2.first6).pow(2).mean()` — **first 6 of 12 dims; last 6 ignored**

**Score formula:**
```
S = 100·d_seg + sqrt(10·d_pose) + 25·B / 37,545,489
```

## 5. Top-5 OPERATIONAL stealth-tech candidates (refined from prompt's seed list)

### 5.1 Faceted-geometry pixel encoding (A1 SABOR / F-117 analog)

**Mechanism.** Per φ1 SABOR audit, 99.27% of pixels are "deeply interior" to the SegNet
argmax decision (stable under ±32-RGB-uint8 noise). Encode the INDEX of the equivalence-
class orbit, not the specific RGB value, for these interior pixels. The argmax-boundary
pixels (~3-5%) need lossless RGB encoding (E4 hybrid).

**Math.** Per-pixel logit-margin `m(p) = logits_top1(p) - logits_top2(p)`. Define
`E_τ(p) = { v ∈ [0,255]^3 : SegNet_argmax(v at p) = GT_argmax(p) }`. For `m(p) > τ`,
`|E_τ(p)|` is large; encode `index_into_E_τ` cheaply.

**Predicted Δscore.** -0.005 to -0.020 (rate + seg-axis combined).

### 5.2 Active cancellation of pose residual (A5 / B-2 antiphase analog)

**Mechanism.** PR101's pose-residual `r_p` (PoseNet output drift) is a structured signal
across 600 pairs. Encode a small `corrections.bin` sidecar that, when XOR/added at
inflate, CANCELS the residual.

**Math.** `r_p = PoseNet(PR101_render).first6 - PoseNet(GT_render).first6`.
Encode `r_p` via arithmetic coding at ~30-60 bytes/pair × 600 = 18-36 KB sidecar.
Pose-axis Δ saturates to zero; rate cost = +18-36 KB → +0.012 to +0.024 rate term.
**Net Δscore = -0.010 to -0.020** if pose-axis marginal compensates.

### 5.3 Decoy targets (B1 Tacit Blue analog)

**Mechanism.** Render baseline frame `f_b` matching expected statistics. Encode deviation
`δ = f_GT - f_b`. The baseline's PoseNet+SegNet output ALREADY matches GT-mean output by
construction; deviation only corrects deviations from mean. Lower entropy.

**Math.** `δ` distribution has variance ~ 0.4-0.6× raw frame variance, so entropy ~ 0.5-0.7×
raw. Net rate savings 12-24 KB. **Predicted Δscore: -0.003 to -0.012**.

### 5.4 Multi-spectral signature reduction (A4 / F1 multi-spectral Lagrangian)

**Mechanism.** Balance SegNet+PoseNet+rate simultaneously via AFRL-style multi-spectral
material design. Sweep the Lagrangian (α_seg, β_pose, γ_rate) across the Pareto frontier.
Select the operating point that minimizes worst-case score under proxy-auth drift.

**Math.** Robust optimization: `min_θ max_(α,β,γ) ∈ Π S(θ; α,β,γ)`. Pareto frontier Π
shrinks operating-point-fragility by 5-15%.

**Predicted Δscore.** -0.006 to -0.025.
