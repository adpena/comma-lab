### Canonical reasoning per ranking

**Rank #1 HILL** — canonical Li-Wang-Li-Huang 2014 high-pass-low-pass aggregation is orthogonal to UNIWARD's wavelet-directional-residual cost because HILL aggregates modification positions in a different way (per Catalog #308 alternative-reducer enumeration discipline). HILL convolves a 3x3 high-pass filter `[-1 +2 -1; +2 -4 +2; -1 +2 -1]` followed by two 3x3 averaging filters, producing per-pixel cost as `cost(i,j) = 1/(L_{i,j} + epsilon)` where L is the smoothed residual magnitude. For canonical per-pair-selector extension: HILL cost interprets pair-pair coupling via the low-pass aggregation step, which is orthogonal to UNIWARD's per-pair scorer-response weighting. Compounding with Slot FF UNIWARD via canonical Pareto polytope intersection per Catalog #372 is mathematically grounded (different axes; canonical Dykstra alternating projections feasible).

**Rank #2 MiPOD** — canonical Sedighi-Cogranne-Fridrich 2015/2016 Fisher-information cost is HARD-EARNED-vs-CARGO-CULTED distinct from heuristic distortion functions; it derives the proper modification probabilities from optimal-detector hypothesis + PLS theory. For canonical per-pair-selector extension: MiPOD cost is `cost(i) = (1/2) * log((1 - beta_i)/beta_i)^2 / sigma_i^2` where `beta_i` is per-pixel modification probability and `sigma_i^2` is per-pixel variance estimate. For per-pair-selector adaptation: the Fisher-information formula generalizes to per-pair-variance estimates derived from canonical scorer-response variance across pairs. Compounding-automation MEDIUM because model-driven cost partially overlaps with UNIWARD's directional-sensitivity per canonical Cauchy-Schwarz META-LIFT analysis; canonical empirical validation required to confirm orthogonal.

**Rank #3 CMD cost polarization** — canonical Tang-Li 2014 non-additive distortion multiplies with ANY additive cost function (UNIWARD/HILL/MiPOD) by polarizing costs based on neighborhood embedding directions; canonical CMD update rule: `cost(i,j) = base_cost(i,j) * polarity_factor(direction_alignment_with_neighbors)`. For canonical per-pair-selector extension: CMD generalizes to per-pair-cluster polarization where pairs in same temporal/spatial cluster receive coordinated cost adjustment. Compounding-automation HIGH because CMD is canonically composable with all rank #1-#5 additive cost functions (canonical multiplication; per Catalog #356 per-axis decomposition preserves orthogonality).

**Rank #4 HUGO** — canonical Pevný-Filler-Bas 2010 SPAM-feature high-dimensional cost is the canonical predecessor to UNIWARD; partly redundant with UNIWARD because UNIWARD descends from the same SPAM-feature lineage (HUGO uses 4th-order SPAM cooccurrences; UNIWARD uses wavelet directional residuals). Compounding-automation MEDIUM per canonical Cauchy-Schwarz META-LIFT analysis suggesting overlap with UNIWARD axis.

**Rank #5 GFR Gabor filter residual** — canonical Song-Liu-Yang-Luo 2015 Gabor 2D filter outperforms DCTR via better local-frequency feature extraction; for canonical per-pair-selector extension: GFR cost interprets pair-pair coupling via Gabor convolution across directions.
