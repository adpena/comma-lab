## 2. The selected path: Path (b) JUSTIFIED-EXTEND with explicit FORK

### 2.1 Layer 1 META-unwind: Atick-Redlich as SINGLE substrate-optimal anchor

**Decision**: The new ATW V2 cooperative-receiver substrate is built FROM ATICK-REDLICH 1990 PRECONDITIONS. The substrate's binding mathematical framework is:

> The encoder + per-pair latent + decoder forms a continuous-time information channel against a FIXED RECEIVER (= contest SegNet+PoseNet). The substrate-optimal loss is L_AR = -I(X; R(decoder(z))) where X = contest input frames, R = receiver = SegNet+PoseNet+EMA-output composition, z = per-pair latent. The mutual-information maximization is subject to (a) rate constraint on archive bytes (Shannon R(D) bound); (b) HNeRV parity non-negotiables; (c) contest-formula composition.

**Demoted theorems** (advisory cross-checks; NOT binding axioms):
- **Tishby IB 2015**: cross-check that compressed I(z;X) preserves I(z;Y_contest) where Y_contest is contest score. Multi-task IB framing (Tishby-Zaslavsky 2015 extensions) MAY apply for per-pair latent regularization. Not in core loss; if used, isolated as advisory regularizer.
- **Wyner-Ziv 1976**: DROPPED. The contest geometry does NOT satisfy WZ preconditions (scorer_class_prior_table is both-sides-shared). Conditional source coding R(D|Y) for known Y on both sides is the correct rate-distortion framework if conditioning is used. The substrate may use conditioning at the rate-axis level (e.g. class-conditional CDF) but it is NOT WZ — it is conditional source coding with known conditioning variable.
- **Schmidhuber compression-as-intelligence 2009**: advisory cross-check that the cooperative-receiver loss IS a compression-as-intelligence formulation (predictability of contest score from compressed bytes); useful for high-level sanity check.

### 2.2 Layer 2 paradigm-binding unwinds

**CC-1 Wyner-Ziv unwind**: scorer_class_prior_table reframed as DETERMINISTIC SHARED CONDITIONING (NOT WZ side-info). The conditioning variable IS in the archive; both encoder + decoder see it. Rate-distortion treatment: R(D|Y) where Y = conditioning variable. The encoder's optimization minimizes Σ_pair rate(z|Y_pair) + distortion(decoded(z|Y_pair)) where decoded includes scorer-conditional gating.

**CC-2 Tishby IB unwind**: IB target is contest score (composite pose+seg+rate). Multi-task IB if used; isolated as advisory regularizer.
