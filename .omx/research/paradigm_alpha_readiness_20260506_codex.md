## Adversarial Review

- This audit is not dispatch evidence. It sets
  `ready_for_exact_eval_dispatch=false` for every row.
- Runtime hook checks are source-string probes, not semantic runtime proofs.
  WMC1/VQM1 still need actual archive builders, charged payload manifests, and
  inflate tests before exact CUDA eval is considered.
- The wavelet byte-negative result should block naive runtime wiring. The next
  wavelet patch should be a byte-positive residual/thresholding profile or a
  new empirical row, not a larger archive path.
- Grayscale-LUT has the most complete runtime path, but it still needs a fresh
  current empirical row and an exact archive candidate proving the grayscale
  distribution consumed at inflate is the one trained or otherwise validated.
